Описание вспомогательных инструментов
-------------------------------------

Для корректной работы вспомогательных инструментов необходимо наличие следующих зависимостей:
 - git;
 - svn, версия не ниже 1.9;
 - java, версия не ниже 1.8;
 - python, версия не ниже 3.4;
 - ant;
 - lcov;
 - cmake;
 - lib32z1;
 - для ручной сборки инструмента CIF: ocaml до версии 4.01 включительно;
 - для сборки инструмента CIL: libmpc-dev (Ubuntu), libmpc-devel (Fedora);
 - для отображения результатов: модуль requests для python3.

1. Инструмент klever — web-интерфейс, отвечающий за визуализацию результатов.
Репозиторий: https://forge.ispras.ru/git/klever.git
Развёртывание web-интерфейса описано в инструкции docs/web_interface.txt.

2. Инструмент clade отвечает за сборку ядра.
Репозиторий clone https://github.com/17451k/clade.git

3. Инструмент CIL отвечает за объединение исходных файлов и их упрощение.
Репозиторий https://forge.ispras.ru/git/cil.git
Примечание: инструмент может быть собран только при наличии ocaml-4.01 или более ранних версий (например, на Ubuntu 14.04 или более ранних версиях) согласно инструкции README.md.
Для более поздних версий ocaml и систем Ubuntu 12.04 (x86_64), Ubuntu 14.04 (x86_64), Ubuntu 16.04 (x86_64) и Fedora 22 (x86_64) есть собранная версия инструмента в архиве tools/cil.xz.
Для установки достаточно его распаковать.

4. Инструмент BenchExec отвечает за ограничение и измерение вычислительных ресурсов.
Репозиторий: https://github.com/sosy-lab/benchexec.git

5. Инструмент CPAchecker отвечает непосредственно за верификацию.
Основной репозиторий: https://svn.sosy-lab.org/software/cpachecker/trunk
Требуемые версии инструмента хранятся в файле cpa.config в корневой директории в формате <ветка> <ревизия>.
Для скачивания конкретной версии необходимо выполнить (из директории tools):
../download_cpa.sh <ветка> <ревизия>
Для установки конкретной версии необходимо выполнить (из директории tools):
../install_cpa.sh <ветка>
Для автоматического скачивания и установки требуемых версий необходимо выполнить (из корневой директории):
./update_cpa.sh
Если требуется обновить ревизию инструмента, то необходимо выполнить (из корневой директории):
cd tools; ../download_cpa.sh <ветка> <новая ревизия>; ../install_cpa.sh <ветка>; cd ..

6. Инструмент CIF требуется для проверки коммитов.
Репозиторий: https://forge.ispras.ru/git/cif.git
